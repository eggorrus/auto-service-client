# Клиентское приложение для управления автосервисом

JavaFX-приложение, позволяющее взаимодействовать с сервером для учета клиентов, мастеров, заказов и цен на услуги. Разработано в рамках курсовой работы.

## Технологии
- Java 17
- JavaFX
- HTTP Client (встроенный)
- Gson

## Запуск
1. Убедитесь, что запущен сервер
2. Запустите приложение:
   - **Через Maven:** `mvn clean javafx:run`
   - **Через IDE:** запустите класс `ui.MainApp`
